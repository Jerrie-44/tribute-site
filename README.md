# Tribute Page: In Loving Memory of EFFIONG, ASUQUO BASSEY

This project is a simple, elegant tribute page dedicated to the memory of Effiong, Asuquo Bassey. It provides a platform for friends, family, and colleagues to share their cherished memories and heartfelt messages.

## Features

*   **Personalized Tribute:** Dedicated to a specific individual with their name and dates of life.
*   **Image Portrait:** Displays a portrait of the honored individual.
*   **Tribute Form:** Allows visitors to submit their name, relationship to the deceased, and a personal message.
*   **Netlify Integration:** The tribute form is configured to work seamlessly with Netlify for easy form submission handling.
*   **Responsive Design:** Styled to be visually appealing and functional across various devices.
*   **Custom Typography:** Utilizes "DM Sans" and "DM Serif Display" for a dignified and readable aesthetic.
*   **Decorative Elements:** Includes subtle background textures and decorative branch images to enhance the visual theme.

## Technologies Used

*   **HTML5:** For the structure and content of the web page.
*   **CSS3:** For styling and layout, including responsive design and custom fonts.
*   **Netlify:** For hosting and handling form submissions.

## Setup and Deployment

This project can be easily deployed to Netlify.

1.  **Clone the Repository:**
    ```bash
    git clone <repository_url>
    ```
2.  **Deploy to Netlify:**
    *   Log in to your Netlify account.
    *   Create a new site from Git.
    *   Connect your repository.
    *   Netlify will automatically detect the `index.html` file and deploy your site.
    *   Ensure your Netlify build settings are configured correctly for form submissions (the `data-netlify="true"` and `name="tribute"` attributes are already set in the form).
